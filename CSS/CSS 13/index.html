Robust

Robust yra žinomas dėl savo galios sunaikinti viską, kas nėra pritvirtinta varžtais.

Mokslinis pavadinimas: Asistentas

Vidutinis aukštis: 220 cm

Svoris: 30–80 kg

Gyvenimo trukmė: 220 metai

Robust – yra pavojingiausias darinys visoje galaktikoje, jie gali sunaikinti ištisas kosmines stotis naudodami tik įrankių dėžę ir ietį, juos perveria paprasta dujokaukė, suteikianti jiems neįtikėtinos jėgos
